# Crop Production Analysis in India (1997–2023)

## Project Overview
This project analyzes crop production trends across different states in India using Power BI. The dashboard provides insights into production, yield, crop distribution, and year-wise agricultural performance.

## Objectives
- Analyze crop production trends across India
- Compare state-wise production and yield
- Identify top-performing crops and states
- Visualize year-wise agricultural growth patterns
- Create an interactive dashboard for data-driven insights

## Tools & Technologies Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Dataset
Source: India Data Portal

## Dashboard Features
- KPI Cards
- Year-wise Trend Analysis
- State-wise Production Comparison
- Crop-wise Analysis
- Interactive Filters & Slicers


## Key Insights
- Certain states contribute significantly higher crop production compared to others.
- Production trends vary across different years.
- Some crops consistently show higher yield performance.
- The dashboard helps identify agricultural patterns and regional differences.

## Conclusion
This project helps understand agricultural production trends in India through interactive visual analysis and supports data-driven decision-making.
